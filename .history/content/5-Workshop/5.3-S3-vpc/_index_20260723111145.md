## Backend foundation

In this section, you create the user-facing API layer for the learning journal application. The frontend authenticates with Cognito, then calls API Gateway routes. API Gateway invokes the main Lambda function, and Lambda stores or retrieves data from DynamoDB and S3.

Core responsibilities:

- Authenticate users with Cognito and protect routes with a JWT authorizer.
- Create a Lambda API handler for health checks, profile information, journal CRUD, image presigned URLs, AI job submission, quiz routes, search, and analytics.
- Create DynamoDB tables for journal logs and AI results.
- Create an S3 bucket for user-uploaded learning images.
