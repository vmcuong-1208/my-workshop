# Create a private S3 access path from a VPC

In this lab, you will show how private workloads inside an Amazon VPC can reach Amazon S3 without going through the public internet. The goal is to create a secure, low-latency path for data access while keeping the environment private.

## What you will learn

- How a gateway endpoint allows EC2 instances in a VPC to reach S3 privately.
- How to create the endpoint, attach it to the correct route table, and verify connectivity.
- How to test S3 access from inside the VPC using the AWS CLI.
