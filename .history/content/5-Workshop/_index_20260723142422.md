# Building an AI Personal Learning Platform on AWS

In this workshop, you will build a serverless learning log application that allows users to log in, create learning logs, upload images, and generate AI summaries, reports, quizzes, search results, and data analytics.

The implementation follows a practical AWS architecture including:

- **Amazon Cognito** for user authentication and Hosted UI login support.

- **Amazon API Gateway + AWS Lambda** for deploying protected APIs and business logic.

- **Amazon DynamoDB** for storing log data, reports, quizzes, and assignments.

- **Amazon S3** for storing images using presigned URLs.

- **Amazon SQS** and Lambda workers for handling asynchronous AI tasks.

- **Groq API** for rapid AI content creation.

- **CloudWatch, SNS, AWS Budgets, CloudFront, and WAF** for monitoring, alerting, and security.

## Workshop Flow

1. **Authentication and API Foundation** – Configure Cognito, create the main Lambda API, and protect routes with JWT authorizers.

2. **Storage and Data Layer** – Create DynamoDB tables, S3 buckets, and routes for logs.

3. **Asynchronous AI Processing** – Create SQS queues, Lambda workers, and AI report/quiz routes.

4. **Search, Deployment, and Operation** – Add search/analytics handlers, deploy the frontend with Amplify, and configure cost and security controls.
