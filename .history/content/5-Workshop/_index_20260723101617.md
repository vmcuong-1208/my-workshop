# Clean up

Congratulations on completing the workshop. You built a serverless AI learning assistant with authentication, protected APIs, image uploads, asynchronous AI processing, monitoring, and cost guardrails.

## Clean up resources

Delete resources in this order to avoid dependency issues:

1. **Amplify app**
   - Open AWS Amplify.
   - Delete the deployed app or disconnect the branch if you no longer need the frontend.

2. **API Gateway**
   - Delete `AI-Personal-Learning-API`.

3. **Lambda functions**
   - Delete `AI-Personal-Learning-Handle`.
   - Delete `AI-Personal-Learning-Worker`.
   - Delete optional search and analytics handlers.

4. **SQS**
   - Delete `ai-report-jobs`.
   - Delete `quiz-jobs`.
   - Delete the DLQ.

5. **DynamoDB**
   - Delete `JournalLogs`.
   - Delete `AiReports`.
   - Delete `Quizzes`.
   - Delete `QuizAttempts`.

6. **S3**
   - Empty the image bucket first.
   - Delete the bucket.

7. **Cognito**
   - Delete the app client, Hosted UI domain, and user pool if they are only used for this workshop.

8. **Monitoring and guardrails**
   - Delete CloudWatch alarms created for the workshop.
   - Delete SNS topics and subscriptions.
   - Delete AWS Budgets if it was created only for this lab.
   - Delete WAF Web ACL if it is not shared with another application.

9. **IAM**
   - Remove custom roles or policies that were created only for this workshop.

## Final check

After cleanup, open the AWS Billing console the next day and confirm there are no unexpected active resources or new charges.
