      <h1>Create the Lambda AI Worker</h1>
      <h4 id=create-the-worker-function>Create the worker function</h4>
      <ol>
        <li>Open <strong>AWS Lambda</strong> and choose <strong>Create
            function</strong>.</li>
        <li>Function name: <code>AI-Personal-Learning-Worker</code>.</li>
        <li>Runtime: Node.js 22.x.</li>
        <li>Choose an execution role with DynamoDB, SQS, and CloudWatch
          permissions.</li>
        <li>Create the function.</li>
      </ol>
      <h4 id=configure-timeout-and-concurrency>Configure timeout and concurrency
      </h4>
      <p>AI calls can take longer than the default Lambda timeout. Configure:
      </p>
      <div class=highlight>
        <pre tabindex=0
          style=color:#f8f8f2;background-color:#272822;-moz-tab-size:4;-o-tab-size:4;tab-size:4;-webkit-text-size-adjust:none><code class=language-text data-lang=text><span style=display:flex><span>Timeout: 30 seconds or more
</span></span><span style=display:flex><span>Reserved concurrency: a small value such as 2-5 during the workshop
</span></span></code></pre>
      </div>
      <p>Reserved concurrency prevents accidental high parallel AI usage and
        keeps cost predictable.</p>
      <h4 id=store-groq-api-key-securely>Store Groq API key securely</h4>
      <p>In <strong>Configuration</strong> > <strong>Environment
          variables</strong>, add:</p>
      <div class=highlight>
        <pre tabindex=0
          style=color:#f8f8f2;background-color:#272822;-moz-tab-size:4;-o-tab-size:4;tab-size:4;-webkit-text-size-adjust:none><code class=language-text data-lang=text><span style=display:flex><span>GROQ_API_KEY=&lt;your-groq-api-key&gt;
</span></span><span style=display:flex><span>GROQ_MODEL=llama-3.1-8b-instant
</span></span></code></pre>
      </div>

      <h4 id=worker-behavior>Worker behavior</h4>
      <p>The worker should:</p>
      <ol>
        <li>Parse each SQS message.</li>
        <li>Mark the target report or quiz as <code>processing</code>.</li>
        <li>Query <code>JournalLogs</code> for the current user and selected
          time range, tags, or topic.</li>
        <li>Build a prompt for Groq.</li>
        <li>Save the AI response to DynamoDB.</li>
        <li>Mark the item as <code>completed</code>, or <code>failed</code> if
          an exception occurs.</li>
      </ol>
