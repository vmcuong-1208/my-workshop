# Extend private S3 access to on-premises networks

This lab shows how to reach Amazon S3 from an on-premises environment through an AWS private connection. Instead of sending requests over the public internet, you will use an interface endpoint and DNS routing so that private workloads can still access S3 securely.

## What you will learn

- How an interface endpoint provides private connectivity for S3 from on-premises networks.
- How Route 53 Resolver and private hosted zones help resolve the S3 endpoint name correctly.
- How to verify connectivity from a simulated on-premises host.
