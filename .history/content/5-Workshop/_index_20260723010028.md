# Build an AI Personal Learning on AWS

#### Overview

In this workshop, you build a serverless learning journal application that lets users sign in, create learning logs, upload images, and generate AI summaries, reports, quizzes, search results, and analytics.

The architecture uses **AWS Amplify**, **Amazon CloudFront**, **AWS WAF**, **Amazon Cognito**, **Amazon API Gateway**, **AWS Lambda**, **Amazon DynamoDB**, **Amazon S3**, **Amazon SQS**, **Amazon CloudWatch**, **Amazon SNS**, **AWS Budgets**, and **Groq API**.

You will implement the workshop in layers:

- **Frontend and authentication** - Host the SPA with Amplify, protect the edge with CloudFront and WAF, and authenticate users with Cognito.
- **API and data** - Expose HTTP routes through API Gateway, process requests in Lambda, store journal data in DynamoDB, and upload images to S3 through presigned URLs.
- **AI workloads** - Send long-running AI report and quiz jobs to SQS, process them asynchronously in a Lambda worker, and call Groq for prompt completion.
- **Operations** - Add cost guardrails, logs, throttling, DLQ handling, and cleanup steps.
