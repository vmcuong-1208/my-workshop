# Clean up the workshop resources

Congratulations on completing the workshop. You have now explored how private networking can be used to connect AWS workloads and on-premises systems to Amazon S3 without relying on the public internet.

## What you should remove

1. Delete the private hosted zone and any Route 53 Resolver rules that were created for the lab.
2. Remove the CloudFormation stacks used to provision the sample network environment.
3. Empty and delete the S3 buckets used during the exercise.
4. Verify that the related VPC endpoints and security groups are no longer needed.

This final step helps keep your AWS account clean and avoids unnecessary charges after the workshop.
